# Задача после рефакторинга: Навигация по договорам

## Что сделать:
Добавить аккордеон `_navGroupHtml` для договоров, аналогичный корпусам/компаниям.

При раскрытии показываются подпункты по типам договоров:
- **Все договоры** (italic, как "все помещения")
- Аренды
- Субаренды
- Подряда
- Услуг
- Купли-продажи
- Обслуживания

Клик по подпункту вызывает `showEntityList('contract', { contractType: 'Аренды' })`, а `showEntityList` фильтрует список на клиенте по `contract_type`.
