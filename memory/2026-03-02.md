## 🆕 Новая цель: Blended Forecast (факт+план)
- Задача: объединить факт (до сегодня) + план (будущие периоды) → прогноз прибыли на конец периода
- Показывать отклонение от плановой прибыли
- Формула: `Projected Profit = (Actual Revenue to date + Planned Revenue future) − (Actual Expenses to date + Planned Expenses future)`
- Deviation = `Projected Profit − Original Planned Profit (full period)`
- **Заблокировано**: `Document_бит_ФормаВводаБюджета` → Доступ запрещён для `odata.user`
- Нужно: 1С admin должен дать права на чтение БИТ.Финанс объектов
