## Анализ: кнопка "+ Создать помещение" не работает в форме ДС

**Сценарий:** Создание ДС (supplement) для договора аренды → блок rent_objects → кнопка "+ Создать помещение" не реагирует.

**Что проверено:**
- Кнопка: `onclick="toggleRentRoomCreate(this, index)"` — в `rent-objects.js:381`
- Функция: ищет `document.getElementById('ro_room_create_' + index)`, если null → `return` (тихо)
- Мини-форма `_roRoomCreateMiniForm(index)` генерируется в ветке `else` (не ЗУ, не legacy)
- Legacy-ветка: `obj.room_name && !obj.room_id` — рендерится БЕЗ mini-form и БЕЗ кнопки
- Статический HTML содержит все элементы (`ro_room_create_` встречается 3 раза — норм)

**Три ветки в renderRentObjectBlock:**
1. `isLandPlot` (ЗУ) — без мини-формы помещения
2. `obj.room_name && !obj.room_id` (legacy) — без мини-формы и без кнопки!
3. `else` (новый формат, room_id есть) — есть и кнопка, и мини-форма

**Наиболее вероятные причины:**
1. **Мобильный UX**: форма открывается но уходит ниже экрана (нужно скроллить) — modal overflow-y:auto
2. **Тихая JS ошибка**: `ro_room_create_0` нет в DOM из-за ошибки при рендере (нет catch, нет alert)
3. **Legacy данные**: если в последнем ДС rent_objects без `room_id` → ветка 2, кнопки нет

**Следующий шаг:** Попросить Валентину проверить Console DevTools при клике. Код НЕ МЕНЯЛСЯ.
